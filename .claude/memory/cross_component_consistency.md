---
name: cross-component-consistency
description: 同一目录下的组件应参考彼此的样式、布局、代码实现，保持风格统一
metadata:
  type: feedback
---

对 `src/components/ImageProcessing/` 及其他含多个组件的目录，修改其中一个组件时应参考同目录下其他组件的实现，保持一致性：

- **样式布局**：按钮颜色、顺序、间距、提示样式等视觉元素应统一
- **代码功能**：相似功能（如复制到剪贴板、图片缩放限制）的实现方式应保持一致
- **交互细节**：loading 状态、错误处理、用户提示等行为模式应对齐

**Why:** 同一功能区域的组件面向相同用户群体，风格不统一会让用户感到割裂，也会增加维护成本。

**How to apply:** 修改某个组件时，先扫一眼同目录下其他组件的模板和样式，必要时一并对齐。
