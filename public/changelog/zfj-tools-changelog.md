# 执法局工具箱更新日志

## 2025年9月29日更新 (v1.2.0)

### 新功能
1. 新增工作照拼接-两宫格模块，支持两张图片拼接
2. 新增工作照拼接-长图模块，支持多张图片拼接成长图
3. 新增图片交换功能，支持左右图片位置交换
4. 新增图片大小控制功能，自动压缩到5M以内

### 功能优化
1. 优化图片拼接性能，减少卡顿现象
2. 优化压缩算法，采用多级质量压缩策略
3. 优化用户界面，添加加载指示器和进度反馈
4. 优化拖拽功能，改为按钮操作更安全可靠

### 问题修复
1. 修复图片拼接时的性能问题
2. 修复多次上传图片后的显示错误
3. 修复横向拼接时图片显示异常问题

### 移除功能
1. 移除音乐播放组件及相关资源文件
2. 移除拖拽交换功能，改为按钮操作
