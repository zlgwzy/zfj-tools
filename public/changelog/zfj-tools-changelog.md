# 执法局工具箱更新日志

## 2026年5月27日更新 (v1.0.3)

### 新功能
1. 新增图片文字标注功能，每张图片可独立设置标注文字和位置（7 个位置）
2. 新增标注样式全局控制（字号 20~35、字体 5 种、颜色选择）
3. 新增长图拼接生成对比图功能，一键预设"处置前/处置后"居中标注
4. 新增艺术字模板：水墨版、鎏金版

### 功能优化
1. 长图拼接输出自动缩放至最长边 2000px 并采用 JPEG 压缩，文件体积大幅减小
2. 标注字号以最宽图片的实际显示尺寸为基准计算，跨图片视觉大小一致
3. 标注样式控件参照四宫格实现，无背景纯文字 + 阴影增强可读性
4. 参数调整自动重新生成长图（300ms 防抖）
5. 复制到剪贴板自动转为 PNG 格式，兼容性更好
6. 生成对比图时自动抑制重复渲染
7. 移除工作照拼接-两宫格模块

### 问题修复
1. 修复长图拼接标注文字未在最终输出中渲染的问题
2. 修复复制到剪贴板因 MIME 类型不匹配失败的问题
3. 修复字体下拉框宽度不足导致选项显示不全的问题

## 2026年5月26日更新 (v1.0.2)

### 新功能
1. 新增艺术字模式，支持在工作照上方叠加艺术字 PNG 图层
2. 新增艺术字模板选择功能，提供夏日版和执法版两套模板
3. 新增艺术字自由拖拽功能，可在图片区域任意移动位置
4. 新增艺术字缩放功能（0.3~3 倍），适应不同版面需求

### 功能优化
1. 艺术字改为模板选择模式，操作更便捷
2. 网格图片从 el-image 改为原生 img，提升导出渲染质量
3. 模板名称支持换行显示，排版更协调

### 问题修复
1. 修复艺术字在 banner 内显示不全和位置调整不精准的问题
2. 修复导出的工作照图片变形问题

## 2025年5月2日更新 (v1.0.1)

### 新功能
1. 新增工作照拼接-两宫格模块，支持两张图片拼接
2. 新增工作照拼接-长图模块，支持多张图片拼接成长图
3. 新增六宫格三行两列布局
4. 新增已占用格子单击替换图片功能
5. 新增空格输入支持
6. 新增图片拖拽排序功能

### 功能优化
1. 四宫格与六宫格之间切换时保留已输入内容
2. 默认字体改为楷体，更符合工作照风格
3. Logo 区域固定宽度，避免布局偏移
4. 图片自动压缩至 5MB 以内，优化导出质量
5. 移除已弃用的 baseUrl 配置，消除 TypeScript 7.0 兼容警告

### 问题修复
1. 修复浏览器缩放导致的图片导出区域缩放问题
2. 修复文本编辑框无法输入空格的问题
