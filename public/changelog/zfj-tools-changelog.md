## 2026年6月11日更新 (v1.0.8)

### 新功能
1. 新增工作照拼接-对比图模块，独立于长图拼接，支持处置前/处置后标注
2. 图片压缩组件新增复制到剪贴板功能，自动缩放超 1600px 大图
3. 长图拼接新增全分辨率无损导出（JPEG quality 1.0）
4. 新增 Windows 部署脚本 deploy.bat 和 macOS 部署脚本 deploy.sh
5. 四宫格/六宫格新增混合模式（艺术字+文字），艺术字默认在 logo 右侧居中显示
6. 显示模式改为三选一（文字/艺术字/混合模式）
7. 更新日志默认展示最新版本，其余版本折叠展示

### 功能优化
1. 长图拼接压缩目标动态计算为「图片数 × 1MB」
2. 输出缩放按图片数计算，每张图约 2000px，质量更高
3. 纵向预览图片宽度 50% 居中，隐藏滚动条
4. 按钮顺序和颜色统一对齐（复制→绿色、导出→蓝色、清空→红色）
5. 复制到剪贴板提示默认始终显示
6. 长图拼接移除生成对比图功能（已拆分为独立对比图模块）
7. 复制到剪贴板图片最长边限制从 1600px 放宽到 2000px
8. 上传图片小于 3MB 时跳过压缩直接读取，提升速度
9. 四宫格/六宫格上传压缩参数调整为 2000px/0.92/≤3MB
10. 图片压缩初始质量 0.85→0.92，目标 ≤500KB→≤1MB
11. 长图拼接上传最长边 2048px→2000px，对比图上传承诺初始质量 0.85→0.92
12. 四宫格/六宫格导出压缩算法优化（精细降质 + 分辨率缩放兜底）
13. 四宫格/六宫格导出渲染倍率 scale 2→3，充分利用 5MB 预算

### 问题修复
1. 修复部署脚本 .user.ini 权限修改失败问题
2. 修复 deploy.bat Windows 路径兼容性问题
3. 修复各组件 Base64 字符串长度与实际字节大小偏差，压缩目标准确生效
4. 修复导航栏和 README 中组件名未同步更新
