## 1. 图片裁剪（4:3）

将任意比例的图片裁剪为标准的 **4:3** 比例。

### 操作步骤

1. **上传图片** — 点击上传区域或 `Ctrl+V` 粘贴图片
2. **调整裁剪区域**
   - **移动**：拖拽裁剪框内任意位置
   - **缩放**：拖拽裁剪框的四个角
   - 保持 4:3 比例，最小 80×60px
3. **操作**
   - **复制到剪贴板**：自动缩放至最长边 ≤2000px，以 PNG 格式复制
   - **导出图片**：JPEG/PNG（与原图格式一致），自动压缩至 ≤5MB

### 注意事项

- 拖动裁剪框时保持 4:3 固定比例，不可自由变形
- 复制到剪贴板仅支持 PNG 格式
- 支持从剪贴板粘贴图片

## 2. 图片压缩（≤1MB）

将图片自动压缩至 **1MB** 以内，方便上传和传输。

### 操作步骤

1. **上传图片** — 点击上传区域或 `Ctrl+V` 粘贴图片
2. **自动压缩** — 上传后自动进行压缩处理
   - 最长边超过 2000px 时自动缩小
   - 自动调整 JPEG 质量（从 0.92 开始逐步降低）
   - 必要时缩小分辨率（0.8 倍缩放后继续降质）
3. **操作**
   - **复制到剪贴板**：以 PNG 格式复制（最长边 ≤2000px）
   - **导出图片**：保存处理后的图片文件

### 注意事项

- 初始压缩质量 0.92，最低降至 0.1
- 如果降质后仍超过 1MB，会缩小图片尺寸（直至 ≥200px 为止）
- 复制到剪贴板仅支持 PNG 格式，文件大小会较压缩图明显增加

## 3. 图片标注

在图片上添加文字标注，支持自定义位置、字号、字体和颜色。

### 操作步骤

1. **上传图片** — 点击上传区域或 `Ctrl+V` 粘贴图片
2. **输入标注文字** — 最多 200 字，支持回车换行
3. **调整样式**
   - **位置**：7 个位置可选（左上、顶部居中、右上、居中、左下、底部居中、右下）
   - **字号**：20~50px
   - **字体**：微软雅黑、宋体、黑体、楷体、仿宋
   - **颜色**：支持自定义颜色和透明度
4. **操作**
   - **复制到剪贴板**：PNG 格式，最长边 ≤2000px
   - **导出图片**：JPEG 格式，自动压缩至 ≤5MB

### 注意事项

- 标注文字的预览效果与导出效果一致（基于图片实际像素计算字号缩放）
- 文字自动换行：超出可用宽度时自动折行
- 支持手动回车换行

## 4. 批量图片标注

上传多张图片和文本配置，自动将文本逐行标注到图片上并打包为 ZIP 下载。

### 操作步骤

1. **上传图片** — 点击上传按钮，可多选图片
   - 图片支持拖拽排序
2. **输入配置文本** — 每一行对应一张图片的标注文字
   - 行数与图片数一致，按顺序一一对应
3. **调整样式**
   - **位置**：7 个位置可选
   - **字体**：5 种中文字体可选
   - **颜色**：支持自定义颜色和透明度
4. **生成并下载** — 点击 "批量标注并下载 ZIP"，自动打包
   - 文件名格式：`批量标注_YYYYMMDD_HHMMSS.zip`
   - 内部图片格式为 JPEG

### 注意事项

- 文本行数应与图片数量匹配
- 图片尺寸较大会自动压缩至 ≤5MB

## 5. 工作照拼接-四宫格/六宫格

将多张工作照片拼接为 2×2（四宫格）或 2×3（六宫格）网格布局，支持文字标题、艺术字和图片标注。

### 操作步骤

1. **上传图片** — 点击每个格子或 `Ctrl+V` 粘贴图片
   - 支持拖拽排序
   - 已上传的图片右上角有删除按钮（鼠标悬停显示）
2. **选择显示模式**

   > **文字模式**
   >
   > - 输入工作动态文本（最多 200 字，支持 4 行）
   > - 调整样式：字号（20~35）、字体、颜色、行首缩进、标题模式
   >
   > **艺术字模式**
   >
   > - 选择艺术字模板（目前支持「美酒名城·画境绵竹 执法版」）
   > - 调整缩放（0.3~3 倍）
   > - 拖拽可自由移动位置
   >
   > **混合模式**
   >
   > - 上方显示艺术字（同艺术字模式）
   > - 下方显示文字内容（同文字模式样式，无 LOGO）

3. **图片标注**（可选）
   - 开启图片标注后可为每张图片单独设置标注文字
   - 调整标注样式：字号、字体、颜色
4. **操作**
   - **复制到剪贴板**：PNG 格式，最长边 ≤2000px
   - **导出图片**：JPEG 格式，html2canvas 以 3 倍渲染，精细压缩至 ≤5MB

### 注意事项

- 图片上传时自动压缩：最长边 ≤2000px，初始质量 0.92，目标 ≤3MB
- 导出时 3 倍高清渲染，图片细节丰富
- 艺术字模式下必须先选择模板才能导出

## 6. 工作照拼接-长图

多张图片水平或垂直拼接为一张长图，支持标注文字。

### 操作步骤

1. **上传图片** — 点击上传或 `Ctrl+V` 粘贴，支持多选
   - 已上传的图片支持拖拽排序
   - 每张图可单独调整缩放比例
2. **选择排列方向**
   - **横向排列**：图片从左到右水平拼接
   - **纵向排列**：图片从上到下垂直拼接（预览更直观）
3. **调整参数**
   - **图片间距**：调整图片之间的间隙
   - **统一尺寸**：开启后所有图片统一为相同宽度
   - **统一宽度**：自定义统一宽度值
4. **标注文字**（可选）
   - 为每张图片单独设置标注
   - 调整标注样式：字号（20~50）、字体、颜色
5. **操作**
   - **复制到剪贴板**：PNG 格式，最长边 ≤2000px
   - **导出图片**：JPEG 格式，压缩目标为「图片数 × 1MB」

### 注意事项

- 图片上传时自动压缩：最长边 ≤2000px，JPEG 0.92
- 自动重新生成：调整任何参数后，长图会在 300ms 防抖后自动重新生成
- 支持生成对比图功能已拆分至独立模块

## 7. 工作照拼接-对比图

两张图片并排或上下排列，自动标注"处置前"/"处置后"，适合展示整治前后对比。

### 操作步骤

1. **上传图片** — 上传两张图片
   - 左侧/上方为「处置前」，右侧/下方为「处置后」
   - 支持 `Ctrl+V` 粘贴
   - **拖拽图片可交换位置**（顺序互换后自动重新生成）
2. **选择排列方向**
   - **横向排列**：左右并排
   - **纵向排列**：上下堆叠
3. **生成对比图** — 点击"生成对比图"按钮
   - 自动标注红色「处置前」「处置后」标签
   - 自动等比例缩放，使两张图在拼接方向上尺寸一致
4. **操作**
   - **复制到剪贴板**：PNG 格式，最长边 ≤2000px
   - **导出图片**：JPEG 格式，压缩至 ≤1MB

### 注意事项

- 切换排列方向或更换图片后自动重新生成
- 标签字体大小根据图片尺寸自动适配
- 图片上传时自动压缩：最长边 ≤2000px，初始质量 0.92
