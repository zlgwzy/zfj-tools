# 执法局工具箱

基于 Vue 3 + TypeScript + Vite 开发的执法局工作工具箱，提供图片处理、文档处理、表格处理等功能，旨在提高执法局工作人员的日常工作效率。

## 功能特性

### 图片处理
- **工作照拼接-四宫格/六宫格** — 2×2 / 2×3 网格拼接，支持文字编辑、艺术字叠加、图片标注
- **工作照拼接-长图** — 多张图片水平或垂直拼接成长图，支持标注、对比图一键生成
- **图片标注** — 单张图片添加文字标注，支持位置/字号/字体/颜色自定义
- **图片裁剪（4:3）** — 将图片裁剪为标准的 4:3 比例，拖拽移动和四角缩放
- **图片压缩（≤500KB）** — 自动压缩图片至 500KB 以内，支持剪贴板粘贴

### 其他功能
- **绵竹地图** — 集成高德地图，查看绵竹市区道路
- **更新日志** — 在线查看版本更新记录

## 技术栈

- Vue 3.4 + TypeScript
- Vite 5
- Element Plus
- Pinia 状态管理
- html2canvas（图片导出）

## 开发环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

## 项目结构

```
src/
  ├── assets/        # 静态资源（图片、字体）
  ├── components/    # 组件
  │   ├── Changelog/ # 更新日志
  │   ├── HomePage/  # 首页布局
  │   ├── ImageProcessing/ # 图片处理相关组件
  │   └── Map/       # 地图组件
  ├── stores/        # Pinia 状态管理
  ├── App.vue        # 根组件
  └── main.ts        # 入口文件
```
